## Featured Software Engineering
### Near-Earth Object ML Recommender
Automated threat detection and tracking for near-Earth asteroids.

- Objective: Replace manual domain-expert labeling with an adaptive ML workflow that highlights potential threats in large, evolving datasets.
- Methodology: Applied SVM and K-fold validation; used correlation-matrix-based indexing for weight assignment.
- Stack: MySQL (model), RabbitMQ (job runner), Bootstrap (view), Flask (controller), Heroku (CI/CD).

Existing approaches often require manual review and do not adapt well as datasets scale. This project focuses on helping astronomers and researchers prioritize objects for investigation by automating first-pass threat recommendations.
